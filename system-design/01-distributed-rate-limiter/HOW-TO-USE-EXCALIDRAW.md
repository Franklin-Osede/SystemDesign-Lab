# üìñ How to Use the Excalidraw Diagram

## üéØ What is the `diagram.excalidraw` file?

The `diagram.excalidraw` file is a **JSON file** that contains all the drawing data for Excalidraw. It's the native format that Excalidraw uses to save diagrams.

---

## ‚úÖ Option 1: Use Excalidraw Web/Desktop App (RECOMMENDED)

### Steps:

1. **Go to Excalidraw:**
   - Visit: https://excalidraw.com
   - OR download the desktop app: https://github.com/excalidraw/excalidraw/releases

2. **Open the file:**
   - Click "Open" or "Load" button
   - Select `diagram.excalidraw` from your project folder
   - The diagram will load with all elements, colors, and text

3. **Edit if needed:**
   - You can modify colors, text, positions
   - Add more elements
   - Export as PNG/SVG for your video

4. **For recording:**
   - You can draw it live following `EXCALIDRAW-SCRIPT.md`
   - OR use the pre-made diagram and just explain it

---

## ‚úÖ Option 2: Use Obsidian with Excalidraw Plugin

### Steps:

1. **Install Obsidian:**
   - Download from: https://obsidian.md

2. **Install Excalidraw Plugin:**
   - Open Obsidian Settings
   - Go to "Community plugins"
   - Search for "Excalidraw"
   - Install and enable it

3. **Import the diagram:**
   - Create a new note
   - Type: ````excalidraw` (three backticks + excalidraw)
   - Paste the content of `diagram.excalidraw`
   - OR use the plugin's "Import" feature

4. **View/Edit:**
   - Click on the diagram to open in Excalidraw editor
   - Edit as needed
   - It will be embedded in your Obsidian note
